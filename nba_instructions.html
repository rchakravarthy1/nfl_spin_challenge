How To Play

The game is very laid back, and you can choose whatever rules you want, but generally the rules go something like this:

  1. The goal is to create the best roster possible
  2. Only one pick can be locked per spin
  3. Labeled positions must be true position picks. "Bench" players can be any position. Check out the roster lists to see their official position.
  4. Optionally, use the notes section to remember which players you picked

Common Rule Variations Include:

  1. Only one selection per team
  2. All-Time edition: Can choose player who played on that team at some point
  3. Active Roster Only: Currently injured players can't be picked
